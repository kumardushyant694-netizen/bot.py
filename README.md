# DARk Aadhar PDF Tool

## Developer: @vikash1178

### Features
- 🔐 Download Aadhaar PDF
- 👑 Owner control panel
- 💰 Credit system
- 🆓 Owner gets unlimited credits
- 📱 Telegram Bot

### Deployment on Railway

1. Create Railway account
2. Deploy from GitHub
3. Add Environment Variable: `BOT_TOKEN`
4. Deploy!

### Owner Commands
- `/owner` - Open owner panel
- `/addcredits <user_id> <amount>` - Add credits
- `/removecredits <user_id> <amount>` - Remove credits
- `/checkcredits <user_id>` - Check user credits
- `/listusers` - List all users
- `/totalcredits` - Total credits summary
- `/resetcredits <user_id>` - Reset user credits
- `/giveall <amount>` - Give credits to ALL users
- `/mycredits` - Check your credits

### User Commands
- `/start` - Start the tool
- `/mycredits` - Check your credits
